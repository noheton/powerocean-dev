# Disclaimer

## Method of Development

This integration was developed by observing network traffic between the official
EcoFlow mobile application and the EcoFlow cloud API using standard HTTP
inspection tools (e.g., a local proxy or packet capture on a device owned by
the contributor).

No EcoFlow application binary was decompiled, disassembled, or reverse-engineered
to produce this software. No proprietary source code, trade secrets, signing
keys, or cryptographic material belonging to EcoFlow was copied or derived from.

All field names, endpoint paths, and protocol structures documented here were
observed in plaintext HTTP/JSON traffic originating from the contributor's own
account and hardware.

## Interoperability Purpose

The sole purpose of this integration is interoperability: enabling Home Assistant
to monitor and control EcoFlow PowerOcean devices that the user already owns,
using the same cloud API that the official application uses. This falls within the
scope of interoperability-oriented software development recognised by applicable
law (e.g., § 69e UrhG in Germany, Art. 6 Software Directive 2009/24/EC in the EU).
